## Overview

- YelpCamp is a full-stack website for users to create and review campgrounds, crafted as part of my learning journey through Colt Steele's web development bootcamp.
- Demo: https://web-dev-bootcamp.onrender.com/

## Features

- User Authentication and Roles: Users can create accounts with different roles, including an Admin role for platform management.
- Campground Management: Users can upload and edit campground data, ensuring data integrity.
- Comment Section: Unlimited comments can be left on each campground page, fostering community discussion.
- Rating and Reviews: Users can rate campgrounds from 1 to 5 stars and leave detailed reviews.
- Google Maps Integration: Google Maps is integrated into each campground page for location visualization.

## Technologies

- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Passport.js
- Mapping: Google Maps API
- Monetization: Google AdSense
- Deployment: Render, MongoDB Atlas, Cloudinary
